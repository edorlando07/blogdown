Problem Statement

This tutorial shows you how to create a Spotify Recommendation Algorithm using the recommenderlab() and arules() packages.

Load Libraries

To get started, first load the libbraries listed below.

# Core & Viz
library(tidyverse)
library(tidyquant)
library(plotly)

# Modeling
library(recommenderlab)
library(arules)
library(arulesViz)

Load Data

The data was previously downloaded and organized. The data was orignally downloaded using Exportify.
